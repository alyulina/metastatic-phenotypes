#### This folder contains the following data:
1. A table with information for each sample in the study (`metadata.csv`);  
2. Clonal barcode counts for all samples (`bc-counts`);
3. A table that contains the total number of clonal barcodes associated with each cell line in a sample (`cell-line_read_counts.csv`), 
as well as a table with estimated cell counts (`cell-line_cell_counts.csv`);
5. Histological (`histology_tumor_area.csv`) and lung weight (`tissue_weight.csv`) data for each cell line from single-cell-line transplant experiments.
